# Features for Solutions

## Feature - buy-back option
Event organizer can repurchase issued tickets
 - Recover unused tickets
 - Re-assign tickets
 - Control secondary market
 - Mass Exchange Tickets for a new Event

## Feature - Group Tickets
Problem:
- Different time activation
- Limit by N
- Can be re-sold to another person (Can be solved with pre-activation stage)
- Name/Surname/Photo/Etc should be stored outside of chain

## Feature - Resell price limitation
Scope: Concert/Stadium
Problem:
- Ticket price changes with the time / number of tickets sold
- Seat selection
- Tickets can be bulk-purchased and re-sold later with the bigger price
Solution:
- Limit the price of re-sell
Complexity:
- Money  off-chain
